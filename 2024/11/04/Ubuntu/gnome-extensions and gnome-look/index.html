具体流程

总体而言, 我们是基于gnome 来完成相关工作的. 其中包括下载 gnome-extensions 等用户插件, 以及更改 gnome 中的系统插件 ubuntu AppindIcators 中的主题来完成.

gnome-extensions

可参考 https://zhuanlan.zhihu.com/p/34608388

我是在火狐里完成相关工作的, 在edge或者google浏览器中流程应该也类似. 按照下面几步做:

一. 安装 sudo apt install gnome-shell-extensions.

非必须. 这只是一个方便的整合包. 如果想要下更多插件还得折腾.

二. 安装 浏览器附加组件与安装本地连接器.

浏览器插件直接在浏览器下即可. 本地附加组件运行sudo apt install chrome-gnome-shell .

浏览器插件和本地的包这两部分呢都要下好. 一开始我没有下本地组件, 折腾了好一会.

三. 下好相关插件和包后, 打开 https://extensions.gnome.org/ , 并点进想要的插件中.

此时所有报错提示都应该已经消失, 下载按钮也已经可用了. 点击按钮, 下载你想要的插件.

下面是一些推荐插件的文章. https://zhuanlan.zhihu.com/p/344554622 https://wzk0.github.io/gne/

gnome-extensions

除了插件之外, 还有一个很好用的东西是gnome-look
https://www.gnome-look.org/browse/

这里面有着各种各样的美化主题, 可以让原来丑陋的UI界面变得像吃了德芙一样丝滑, 甚至还能把你的电脑伪装成MacBook. 非常好用.

过程比折腾extensions简单.

一. 点开官网, 选择想要的theme或icons, 下载.

二. 将theme文件移动到~/.themes, icons移动到~/.icons. 没有的就先创建好.

三. 打开tweaks, 在Appearance中选用喜欢的主题. (没有tweaks的运行 sudo apt install gnome-tweaks 下载, 或者直接使用 extension manager, 在里面找到 ubuntu AppIndicators 进行设置也行)

总结

至此两个部分都折腾完毕了. 设置完成之后的ubuntu用着丝滑无比, 工作的时候心情也好了不少.

除了这些, 我还进行了终端的外观美化(变成半透明), 壁纸的自动轮换等, 感兴趣的可以自行google.

相关的过程很是花了一番功夫, 但是配置好之后的使用体验飞跃性地提升了. 难怪苹果之类的公司花那么多钱来设计UI界面. 一个好的UI界面和可视化环境真的会让人很舒服, 并且提高你的工作效率和积极性. 建议大伙都折腾折腾.
