华硕天选三从win11换系统到Ubuntu22.04记录

2024-11-02 2024-11-03
WorkLog

换系统配环境的折腾之旅

!!!注意!!!
下面的内容中, 有绝大部份(如果不是全部)都来自ChatGPT生成, 本人按照上面的指示解决了问题, 但并不保证其正确性. 因此, 对于那些大段而有条理性的内容, 请自行甄别其对错!!!

环境如下
本机: 华硕天选三
原系统: windows 11
新系统: Ubuntun 22.04

简而言之, 本人在配环境的时候不小心把window搞坏了, 刚好最近开发的时候总是遇到各种包不支持windows的痛点. 于是痛定思痛, 决定干脆重装系统换用linux.

上午11点: 下定决心重装系统, 开始备份各种重要文件. 浏览文件的时候才发现, 电脑上99%的文件都是不需要带走的, 类似搬家时可以丢弃的垃圾. 对于各种编写的代码和学习的资料, 平时就已经备份到了夸克网盘上; 对于各种环境, 本来也就无法带走. 到头来, 只备份了一些图片和文档. 其他几百G的游戏和软件都打算丢弃.

下午2点: 正式开始着手换系统. 根据GPT的指导开始下载Ubuntu镜像源, 并搞好了引导盘.

之后按部就班地走:

  1. 重启并进入BIOS中完成相关设置: 重启时, U盘需要插在电脑上, 并且最好在重启开始的时候就按住F2不放, 直到进入 BIOS中. BIOS内, 将引导盘调整至原本BIOS的上面, 使得系统启动的时候优先通过引导盘来启动. 此为华硕的操作, 其他机器上具体按键可能不一样.
  2. 设置完成后, 退出并重启, 进入Ubuntu系统安装界面: 根据引导程序按部就班一路确认即可.

下载好本体系统这一点没什么好说的, 不会遇到什么问题.

* 值得一提的是, 一开始我把镜像文件下错了. 本来应该下 Ubuntu Desktop 版本的, 但是我以为 Ubuntu Sever 才是我想要的. 结果一进去对着命令行窗口傻眼了. 只好借朋友的电脑重新刻录了一遍引导盘, 把命令行系统换成了图形化界面的系统.

装上系统本体很简单, 接下来就是重头戏了. 我开始了各种环境的配置与软件的安装. 具体罗列如下:

  1. 中文输入法的安装. 这一点花了很多时间, 网上有许多教程, 但是按照教程做下来却并不管用. 最终下了Fcitx5, 并折腾了好一会才解决了问题. 顺便还配了一个好看舒服的输入法主题 😄
  2. clash的下载. 没花什么功夫. clash本身就支持linux. 但是帮助却很大, 例如没有科学上网的能力, 没有办法用GPT, github很多时候都登不上去, apt的更新也会报错"网站无法访问". 在拥有clash, 并更改了apt工具所使用的镜像源之后, apt终于可以正常使用了.
  3. edge, google 的下载. 根据官网的 .deb文件直接安装完成.
  4. vscode, vim等工具的下载. 直接完成, 没有什么困难.
  5. pycharm下载. 首先得到了一个压缩包, 解压之后出现了一个引导文件. 于是开始通过引导文件的指导进行操作. 总体上压缩包里的东西就是全部了, 可以通过一个命令行直接运行. 之后登陆了帐号获取通行许可, 然后把设置快捷的启动方式, 并添加环境变量 export PATH="$PATH:/home/username/pycharm-2024.2.4/bin". 这个Path需要根据自己的文件位置来加.
