Ubuntu22.04 截图与录屏

2024-11-02 2024-11-01
Note

在 Linux 中，可以使用多种方法进行屏幕截图，具体取决于你使用的桌面环境。以下是一些常见的屏幕截图工具和方法：

1. 使用键盘快捷键

  • 全屏截图：

    • 按 Print Screen（有时标记为 PrtSc）键。这会将整个屏幕的截图保存到你的主目录中的“图片”文件夹。

  • 当前窗口截图：

    • 按 Alt + Print Screen。这将截取当前活动窗口并保存到你的主目录中的“图片”文件夹。

  • 选择区域截图：

    • 按 Shift + Print Screen 或 Ctrl + Shift + Print Screen。这会让你选择要截图的区域。

2. 使用 Screenshot 工具

大多数桌面环境（如 GNOME、KDE、XFCE）都自带截图工具。

  • GNOME Screenshots：

    • 可以在应用程序菜单中找到“截图”或“Screenshot”工具。打开它后，你可以选择截取整个屏幕、当前窗口或自定义区域。

  • KDE Spectacle：

    • KDE 桌面环境使用 Spectacle 作为截图工具。打开它后，可以选择不同的截图选项，并可以对截图进行编辑和注释。
